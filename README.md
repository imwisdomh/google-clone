# Google Clone

## 개요
Google의 메인 페이지를 클론 코딩하여 HTML, CSS, JavaScript의 기본 지식과 활용 능력을 학습하고 적용한 프로젝트입니다.

## 데모
![google-clone](https://github.com/user-attachments/assets/5c2cbb36-b483-48bd-ba10-6ea9586428ed)

## 주요 기능
- Google 검색 기능 (GET 요청)
- 반응형 디자인
- 다크 모드 지원
- 프로필 이미지 변경 기능

## 사용 기술
- HTML5, CSS3, JavaScript
- Font Awesome (아이콘)
- LocalStorage (데이터 저장)

## 프로젝트 구조
```
google-clone

├─ css

│  ├─ light.css
```
